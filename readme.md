# Lumen MongoDB Docker
The project is a basic example of using lumen with mongoDB under docker. And jenssegers/laravel-mongodb is applied.

## Info
Framework: Lumen 5.8
Web: nginx:1.10
PHP: php7.1-fpm
DB: MongoDB 4.0.9 

## Reference

https://github.com/jenssegers/laravel-mongodb#installation
